##### TSO: time sharing option #####
an interactive environment on IBM mainframes (like z/OS) that lets multiple users
simultaneously log in and share system resources, acting like a command line interface

### Managing Data Sets ###
allocate: command that can be used to set up dataset
rename: command for renaming the dataset

ISPF: interactive system productivity facility
    - menu-driven interface on IBM mainframe that provides tools for editing 
    datasets, edit code, develop applications, submit batch jobs, and 
    perform system maintenance

listds: list dataset (shows properties)
listalc: show all files allocated in mainframe
help: show list of all commands available
logoff: current user exits the TSO session
cntl + a: attention interrupt key

### TSO OPERATING MODES ###
foreground mode: interact directly witht he TSO using commands
background mode: TSO commands are run independent of the user

when running the "edit" command, there are subcommands like:
list: show content in dataset
press "enter" and enter data
type "save"
type "list" to view the data set
type "end" to exit the edit mode
